
# gaf

Draw ASCII bar charts with a file


## Installation

### go get

```go
∃! go get github.com/Xx0w0wxX/gaf/cmd/gaf
```

## Usage

```go
∃! cat sample.csv
python,64
go,18
flutter,57
javascript,97
html,58
css,59
php,74

∃! gaf sample.csv
    python :  █████████████████████▎
        go :  ██████
   flutter :  ███████████████████
javascript :  ████████████████████████████████▎
      html :  ███████████████████▎
       css :  ███████████████████▋
       php :  ████████████████████████▋
```
